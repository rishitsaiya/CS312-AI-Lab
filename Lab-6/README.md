## Lab-6

#### Prerequisites

The packages `pandas`, `sklearn`, `tqdm` are needed to run this code locally. Other alternative is to use `12.ipynb` on Google Colab/Jupyter Notebook.

Packages can be installed using:

```sh
pip3 install pandas
pip3 install sklearn
pip3 install tqdm
```

#### How To Run

```
python3 12.py
```
